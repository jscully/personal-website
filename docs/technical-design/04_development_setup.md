# Local Development Setup

## 14. Local Development Setup

### 14.1 Prerequisites
- Node.js 16+ and npm/yarn
- Git

### 14.2 Setup Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/personal-website.git
   cd personal-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Admin Portal Setup** (if running separately):
   ```bash
   # Assuming admin is a separate route or build target
   # Follow specific instructions for Admin module
   ```
